# 海龟汤（情境猜谜）规则

### 什么是海龟汤 / 情境猜谜？

**情境猜谜**（Situation Puzzle），在中文社区常被称为**海龟汤**，是一种多人参与的水平思考游戏。它通常涉及一个神秘、有时甚至恐怖的故事，但最初只揭示故事的一部分。

### 角色

1.  **主持人（DM / 汤主）：**
    *   知道完整的故事（即“汤底”）。
    *   向玩家展示一个简短的、令人困惑的情境（即“汤面”）。
    *   回答玩家的问题。

2.  **玩家（侦探 / 喝汤人）：**
    *   聆听情境。
    *   向主持人提问，以揭示隐藏的细节并重构完整的故事。

### 游戏流程

1.  **出题：** 主持人朗读情境（谜题陈述）。
2.  **调查：** 玩家向主持人提问。
3.  **反馈：** 主持人回答问题。
4.  **解答：** 当玩家收集到足够的信息后，他们提出一个解决方案（完整故事）。如果正确，游戏结束。如果不正确，调查继续。

### 提问规则

游戏最关键的规则是对提问的限制。玩家通常只能提出可以用以下方式回答的问题：

*   **是**（问题中的假设是正确的）
*   **不是**（假设是不正确的）
*   **是也不是**（问题包含正确和错误的成分，或者在某种意义上正确但另一种意义上不正确）
*   **无关**（问题对解谜没有影响）

*注：在某些变体中，如果简单的“是”或“不是”会产生误导，主持人可能会提供简短的澄清，但严格遵守这四种回答是标准玩法。*

### 示例

*   **汤面：** 一个男人走进一家酒吧，要了一杯水。酒保拿出一把枪指着他。男人说了一句“谢谢”然后离开了。为什么？
*   **玩家：** 那个男人渴了吗？
*   **主持人：** 否。
*   **玩家：** 枪是真的吗？
*   **主持人：** 是。
*   **玩家：** 那个男人害怕吗？
*   **主持人：** 是（起初）。
*   **汤底（真相）：** 那个男人在打嗝。酒保看出来了，于是用枪吓他来治好打嗝。男人意识到打嗝好了，所以感谢了酒保。
